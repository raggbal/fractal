# 0.195.641

## 概要
0.195.637 (balanced paren parser) に加え、Notes / Outliner の side panel md editor でリンク挿入 (Cmd+/ → link) が動作しない問題を修正。

## 課題
Notes editor → outliner → side panel で .md ページを編集中に Cmd+/ → リンク挿入を選択しても何も起きない。同様に .out standalone → side panel でも動作しない。.md を直接 Fractal Editor で開いた場合のみ動作する。

## 原因

### 原因 1: host bridge に requestInsertLink メソッドが未実装
`outliner-host-bridge.js` と `notes-host-bridge.js` に `requestInsertLink` メソッドが存在しなかった。`vscode-host-bridge.js` にのみ実装されていたため、.md 直接開き以外では呼び出しが silent fail。

### 原因 2: host handler に insertLink メッセージの処理がなかった
`outlinerProvider.ts` と `notes-message-handler.ts` に `case 'insertLink':` がなく、webview からのリクエストが受け付けられなかった。

### 原因 3: outliner.js の setupHostMessages に insertLinkHtml 転送ケースがなかった（根本原因）
Notes webview では editor.js の main instance は作成されない (`__SKIP_EDITOR_AUTO_INIT__`)。host からの postMessage は outliner.js の `setupHostMessages` が先に受け取る。`insertImageHtml` は outliner.js:4750 で side panel に転送していたが、`insertLinkHtml` の転送ケースが欠落していたため、side panel editor に host のレスポンスが届かなかった。

## 対策

### 設計
既存の画像挿入 (`insertImageHtml` / `requestInsertImage`) パターンに完全に倣い、3 箇所に実装:

1. **host bridge** — リクエスト送信メソッド
2. **host handler** — `showInputBox` ダイアログ表示 → `insertLinkHtml` レスポンス送信
3. **outliner.js setupHostMessages** — `insertLinkHtml` を side panel editor に転送

### 実装

| ファイル | 変更 |
|---|---|
| `src/shared/outliner-host-bridge.js` | `requestInsertLink` メソッド追加 |
| `src/shared/notes-host-bridge.js` | `requestInsertLink` メソッド追加 |
| `src/shared/sidepanel-bridge-methods.js` | `requestInsertLink` メソッド追加 |
| `src/outlinerProvider.ts` | `case 'insertLink':` handler 追加 (`t('enterUrl')` / `t('enterLinkText')` で i18n 対応) |
| `src/shared/notes-message-handler.ts` | `NotesPlatformActions` に `requestInsertLink` 追加 + `case 'insertLink':` handler 追加 |
| `src/notesEditorProvider.ts` | `requestInsertLink` platform 実装 |
| `src/webview/editor.js` | `SidePanelHostBridge.requestInsertLink` を `_mainHost` に委譲 + `sidePanelLinkPending` フラグ + `onLinkRequest` コールバック |
| `src/webview/outliner.js` | `case 'insertLinkHtml':` を `setupHostMessages` に追加 (side panel 転送) |
