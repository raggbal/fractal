# Release v0.195.674

**リリース日:** 2026-04-13
**バージョン種別:** patch
**対応要件:** requirement v9 / design v9

## 概要

MD editor (side panel 版) で画像リンク `![](path)` やファイルリンク `[📎](path)` をコピー＆ペーストする際、別 outliner / 別 note の MD に貼り付けた場合にリンク先の物理ファイルも自動複製する機能を追加。同一 outliner 内のペーストではテキストのみ貼り付け（既存動作維持）。

## 変更内容

### 新機能
- **FR-1: クリップボードコンテキスト**: copy/cut 時に `text/x-any-md-context` として sourceImageDir / sourceFileDir / sourceMdDir（絶対パス）をクリップボードに保存
- **FR-2: ペースト判定**: paste 時にソースとデスティネーションのディレクトリを比較し、異なる場合のみアセット複製を実行
- **FR-3: 画像アセット複製**: 画像は `copy-{timestamp}-{originalName}` で rename コピー
- **FR-4: ファイルアセット複製**: ファイルは元名保持 + collision suffix (`-1`, `-2`, etc.)
- **FR-5: 同期型フロー**: paste → host (pasteWithAssetCopy) → ファイルコピー + パス書き換え → pasteWithAssetCopyResult → DOM 挿入
- **FR-6: 全ケース対応**: 自MD内（テキストのみ）、同outliner他MD（テキストのみ）、別outliner/別note（アセット複製）
- **FR-7: copy/cut 統一動作**: どちらでもファイルは常に複製（ソース削除なし、cleanup に任せる）
